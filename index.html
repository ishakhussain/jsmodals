The Fundamentals of Front-end Development

What is HTML?

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for Web pages
  • HTML elements are the building blocks of HTML pages
  • HTML elements are represented by < > tags

What is CSS?

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media.
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once.
  • It allows you to set background colours and fonts, and laying out content in multiple columns, as well as a whole lot more!

What is Javascript?

  • Javascript is the Programming Language for the Web.
  • Javascript can update and change both HTML and CSS.
  • Javascript can calculate, manipulate and validate data.
  • It also enables you to create dynamically updating content, control multimedia, animate images, and pretty much everything else.
